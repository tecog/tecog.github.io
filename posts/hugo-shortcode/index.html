使用Hugo Shortcode引用代码文件

2024.04.15

偶遇一需求，需要在hugo文章中引入代码文件，而不是直接粘贴代码，这样可以保持代码文件的原貌，同时也方便维护。Hugo提供了shortcode功能，可以方便的实现这个需求。

在主题或项目的 layouts/shortcodes 目录下创建一个新的文件，文件名即为 shortcode 的名字，例如 code.html，然后在文件中编写 shortcode 的内容。

{{ $file := .Get "file" | readFile }}
{{ $lang := .Get "lang" }}
{{ $escapedFile := htmlEscape $file }}
{{ $escapedCode := printf "<pre><code class=\"%s\">%s</code></pre>" $lang $escapedFile | safeHTML }}
{{ $escapedCode }}

在文章中引用content/codes/example.c

{{< code file="codes/example.c" lang="C" >}}

效果如下 

#include <stdio.h>
int main() {
    return 0;
}
