Live2D 看板娘控件添加

2022.12.13

偶然间在别人的博客内看到了右下角的可爱少女，很神奇的是她的目光会跟随鼠标位置，活灵活现，于是研究一番，成功在博客上部署了Live2d.

配置代码

themes/zozo/layouts/partials/js.html

<script type="text/javascript" charset="utf-8" async="" 
    src="https://cdn.jsdelivr.net/npm/live2d-widget@3.0.4/lib/L2Dwidget.min.js">
</script>
<script type="text/javascript">
    setTimeout(() => {
    L2Dwidget.init({
        "model": {
            "scale": 1
        },
        "display": {
            "position": "right",
            "vOffset": -70
        },
        "mobile": {
            "show": false,
            "scale": 0.2
        },
        "react": {
            "opacityDefault": 0.7,
            "opacityOnHover": 0.2
        },
        "dev": {
            "border": 0
        }
    });
}, 1000)
</script>
